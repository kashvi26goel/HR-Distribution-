# HR Analytics Dashboard

## Overview

The HR Analytics Dashboard is a comprehensive data analytics project designed to analyze workforce demographics, employee retention, diversity, turnover trends, and organizational performance using **SQL**, **Power BI**, and **data visualization techniques**.

This project transforms raw HR data into actionable insights that help organizations improve employee retention, workforce planning, diversity initiatives, and strategic HR decision-making.

---

## Project Files

### `project 3(1).pbix`

Power BI dashboard containing:

* Interactive HR analytics visuals
* Employee demographic insights
* Turnover analysis
* Diversity metrics
* Workforce distribution analysis

### `project.sql`

SQL queries used for:

* Data cleaning
* Data transformation
* KPI generation
* Aggregation and analysis

### `Human Resources.csv`

Main HR dataset containing employee details.
